# Pensioenplanner

Een zelfstandige pensioenplanner als één HTML-bestand. React, ReactDOM en Babel worden via een CDN geladen. Er is geen build-stap nodig.

Invoer wordt lokaal in de browser opgeslagen (localStorage). Er gaat niets naar een server.

## Lokaal openen

Open `index.html` direct in een browser.

## Online zetten

Zie GitHub Pages in de instellingen van de repository. De site verschijnt op `https://<gebruikersnaam>.github.io/<repo-naam>/`.
